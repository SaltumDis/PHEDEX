PhEDEx Widget examples

See https://twiki.cern.ch/twiki/bin/view/CMS/PhEDExWidgetLibrary for the accompanying documentation. These examples are coded with the assumption that you have the YUI toolkit installed locally, and are running the PhEDEx proxy-server configured to use it. See https://twiki.cern.ch/twiki/bin/view/CMS/PhEDExWebsiteDeveloperGuide for the documentation about that.

Source code for these examples is in CVS in *PhEDExWeb/ApplicationServer/examples*

  • Core widget with simple manipulation of its DOM elements.
  • Derived widget with sample 'Extra' information and a delayed (but fake) 'update' call.
  • Simplified DataTable widget which implements the API 'Nodes' call.
  • Simplified TreeView widget with random data.
  • TreeView with dynamically loaded data.
  • ContextMenu with simple example of use of registration.
  • A TreeView with a ContextMenu which calls out to other widgets.
  • A ContextMenu handler which serves anything that can give it a node-name.
  • Control-elements and a few variations thereon.
  • Data look up widget used to display block information.
  • History demo used to demo history in the widget states.
  • Group usage widget used to display group information.
