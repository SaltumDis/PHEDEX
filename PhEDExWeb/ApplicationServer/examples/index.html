PhEDEx Javascript examples

See https://twiki.cern.ch/twiki/bin/view/CMS/PhedexProjWebsite for the accompanying documentation. These examples are coded with the assumption that you have the YUI toolkit installed locally, and are running the PhEDEx proxy-server configured to use it. See https://twiki.cern.ch/twiki/bin/view/CMS/PhEDExWebsiteDeveloperGuide for the documentation about that.

Source code for these examples is in CVS in PhEDExWeb/ApplicationServer/examples

  • Use the Loader to load a YUI class and do something when it arrives
  • Use the Logger with access to the log-to-server controls
  • Use the Profiler with a few sample objects churning up CPU
  • Load a complete but empty application i.e. with the core, sandbox, logger, profiler, everything except the parts that rely on PhEDEx data itself
  • Load the "Nodes" module with the full core/sandbox environment to create it
