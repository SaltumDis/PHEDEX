Welcome to the PhEDEx Project homepage. PhEDEx is the name given to the system of Agents and the central Transfer Management Database (TMDB) developed by CMS for the 2004 data challenge. It is used for scheduled transfers of CMS Physics data.

These pages contain documentation of the system and related talks, links to savannah (bug tracking), CVS and the mailing list archives. To join the list log into Simba.
